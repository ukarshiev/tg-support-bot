## Graphify

В проект подключён Graphify: локальный граф знаний лежит в `graphify-out/` и помогает быстрее понимать связи между файлами, классами, функциями и маршрутами.

Когда пользователь пишет `/graphify`, сначала используй Graphify, а уже потом обычный поиск по коду.

Правила работы:
- Для вопросов по устройству проекта сначала запускай `graphify query "<вопрос>"`, если есть `graphify-out/graph.json`.
- Для поиска связи между двумя сущностями используй `graphify path "<A>" "<B>"`.
- Для объяснения одной сущности используй `graphify explain "<название>"`.
- Изменения в `graphify-out/` после хуков или обновления графа — это нормально.
- После изменения кода запускай `graphify update .`, чтобы граф не устаревал.
